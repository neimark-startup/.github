# Удаление папки .idea из Git репозитория

## 1. Удаление папки .idea из репозитория

```bash
# Удалить папку .idea из индекса Git (но оставить локально)
git rm -r --cached .idea

# Создать коммит с удалением
git commit -m "Remove .idea directory from repository"
```

## 2. Настройка .gitignore

Отредактируйте файл `.gitignore` в корне репозитория:

```gitignore
# IDE
.idea/
*.iws
*.iml
*.ipr

# JetBrains IDE
*.swp
*.swo
*~
```


## 3. Проверка статуса

```bash
# Убедиться, что .idea больше не отслеживается
git status
```

## 5. Для предотвращения в будущем

- Убедитесь, что `.gitignore` добавлен в корень репозитория
- Проверяйте `git status` перед коммитом
- Используйте `git add .` с осторожностью, лучше добавлять файлы явно

## Важно!
Папка `.idea` останется на вашем локальном компьютере, но перестанет отслеживаться Git.
